# 💄 Glow Beauty Sales Dashboard

## 📌 Project Overview

The **Glow Beauty Sales Dashboard** is an interactive Power BI dashboard created to analyze sales performance, customer purchasing behavior, and product trends for a beauty and skincare business. This project provides business insights through interactive visualizations and KPIs to support data-driven decision-making.

---

## 🎯 Objectives

- Analyze overall sales performance.
- Track revenue and order metrics.
- Identify top-selling products.
- Monitor product category performance.
- Visualize customer purchasing trends.

---

## 📊 Dashboard Features

### 📈 Key Performance Indicators (KPIs)

- 💰 Total Revenue
- 🛒 Total Orders
- 📦 Products Sold
- 💵 Average Order Value
- 📦 Average Products per Order

### 📊 Visualizations

- Total Orders by Weekday
- Orders by Hour
- Top 5 Selling Products
- Revenue by Product Category
- Monthly Revenue Trend

### 🎛 Interactive Filter

- Product Size
