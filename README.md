# ✨ Glow Beauty Sales Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard created to analyze Glow Beauty sales data. The dashboard provides insights into revenue, customer behavior, product performance, and sales trends to support data-driven business decisions.

## 📊 Dashboard Features
- Total Sales & Revenue KPIs
- Sales Trend Analysis
- Product Category Performance
- Customer Insights
- Interactive Filters & Slicers
- Dynamic Visualizations

## 🛠 Tools & Technologies
- Power BI
- Power Query
- DAX
- Microsoft Excel

- Data Cleaning using Power Query
